Learn from data

At university, I was able to gain experience in the field of Data Science. For a Data Science project I had to use several Python libraries and frameworks such as Numpy, Seaborn, Pandas or Scrapy and save the collected data in a SQLite database before analyzing everything in Jupyter notebook with Pandas and Numpy. The conclusion: Data science is very similar to marketing. You learn from data and draw your decisions from it.

Python and the web

Besides the data science projects, I used Python for other kinds of projects at university: small programs as well as larger ones. My favourite is still my Pyfeed project, in which Python is used on the web with the Flask framework.
